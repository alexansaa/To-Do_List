# World Tour Concerts <a name="about-project"></a>

**To-Do List** is a HTML, CSS and JavaScript project that displays a friendly interface for tracking activities. This project is meant to be a support for your daily live and can be used by anyone how is looking for a simple way to track its progress.

### Key Features <a name="key-features"></a>

- **Webpack implementation for project development**
- **HTML template to render the main page**
- **Dynamic rendering of static activities with the use of an array of JSON objects**
- **Bootstrap and CSS styling**
- **Completed and not completed activities personalized styling**
- **Add, edit & remove activities functionality**
